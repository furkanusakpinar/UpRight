# 🆙 UpRight

**A premium, minimalist posture tracking assistant designed to protect your spinal health.**

UpRight is a modern mobile application built with React Native and Expo that uses your device's accelerometer to track your posture in real-time. Whether your phone is in your pocket, hand, or a chest pocket, UpRight ensures you maintain a healthy spine through intelligent calibration and persistent notifications.

---

## ✨ Core Features

**Real-time Tracking:** High-precision posture monitoring using device accelerometer sensors.

**Smart Calibration:** Calibrate the app based on your current position (pocket, hand, etc.) to set your unique "perfect posture" reference.

**Persistent Notifications:** Stay protected even when the app is in the background with high-priority Android notifications.

**Premium Dark Design:** A sleek, modern UI with an Indigo/Zinc color palette for a premium feel.

**Minimalist Architecture:** Optimized codebase with zero boilerplate, written in pure JavaScript for maximum performance.

---

## 🛠️ Technology Stack

**Frontend:** React Native, Expo Router

**Sensors:** Expo Sensors (Accelerometer)

**Notifications:** Expo Notifications, Task Manager

**Styling:** Native StyleSheet, Expo Linear Gradient

**Icons:** Lucide React Native

## 📱 Getting Started

Follow these steps to run UpRight in your local environment:

<div align="left">

```bash
# Clone the repository
git clone https://github.com/furkanusakpinar/UpRight.git

# Navigate to project folder
cd UpRight

# Install dependencies
npm install

# Start the application
npx expo start
```

</div>

---

## 🧘‍♂️ How It Works

1. **Position Your Phone:** Place your phone in your pocket, hand, or against your chest.
2. **Calibrate:** Stand in your best upright posture and press the **"Calibrate"** button.
3. **Start Tracking:** Press **"Start"**. UpRight will now monitor your tilt relative to your calibrated baseline.
4. **Stay Notified:** If your posture deviates from the baseline for too long, you'll receive a gentle notification to straighten up.
